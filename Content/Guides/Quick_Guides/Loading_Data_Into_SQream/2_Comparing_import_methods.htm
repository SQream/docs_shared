Comparing import methods

The following table summarizes the advantages and limitations of each method.

Command

Advantages

Limitations

COPY FROM

  • Easy to use
  • Can skip rows, such a header row as part of command
  • Can perform date format transformations as part of command
  • Error handling options
  • Local data only (or accessible locally via NFS)
  • Cannot perform any data manipulations other than date format

External Tables
  • Most powerful method
  • Full range of data transformations and manipulations
  • Local data only (or accessible locally via NFS)
  • No error handling options

Data Streaming

  • Can access remote data
  • Can insert directly from another database
  • Requires external data manipulations before streaming
  • Requires 3rd party tool for ETL with SQream connectors or coding
