Query execution tree

To see the query execution tree, use the report () utility function with the relevant path and query as the arguments.

This utility function does not execute the query, it only compiles it.

The output of this report will be a query.png image, with the relevant tree.

select report('/tmp',$$select a,b,c from tableA a join tableB b on a.id=b.id and a>4000 and b like '%this is a sample query%' $$);

If you need help in understanding the report, please contact SQream support at support@sqream.com
