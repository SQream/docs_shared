Session statistics

To check active session statistics and progress, use the following utility function while connected to the cluster (in any database/username):

select show_node_info(statement_id);

  • To find out the statement_id, use the show_server_status utility function.
  • To see the statements statistics for statements that have already finished executing, refer to the log clientLogger_execution%.

For more information on SQream logs, see Admin Guide - Logs.
